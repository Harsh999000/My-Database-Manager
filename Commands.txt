Start spring boot app in background:
- nohup mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Dspring.datasource.url=jdbc:mariadb://your_ip:port/mysql -Dspring.datasource.username=user_name -Dspring.datasource.password=password_here" > spring.log 2>&1 &

Confirm is spring boot app is running:
- ps aux | grep java

Start cloudflare tunnel in background:
- nohup cloudflared tunnel --url http://localhost:8080 > cloudflared.log 2>&1 &

Confirm cloudflare is running:
- ps aux | grep cloudflared

Check for link generated by cloudflare:
- grep "Visit" cloudflared.log
- grep -A 2 "Your quick Tunnel has been created!" cloudflared.log
- Output: Your quick Tunnel has been created! Visit it at (it may take some time to be reachable):
https://what-enzyme-license-they.trycloudflare.com

Stop spring boot app:
- pkill -f "mvn spring-boot:run"

Stop cloudflare tunnel:
- pkill -f "cloudflared"
